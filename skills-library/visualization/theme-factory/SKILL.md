# Theme Factory Skill

This skill provides a curated collection of professional font and color themes, each with carefully selected color palettes and font pairings. Once a theme is chosen, it can be applied to any artifact.

## Purpose

To apply consistent, professional styling to presentation slide decks or other artifacts. Each theme includes a cohesive color palette and complementary font pairings.

## Usage Instructions

1. **Ask for their choice**: Present the available themes and ask which one to apply.
2. **Wait for selection**: Get explicit confirmation about the chosen theme.
3. **Apply the theme**: Once confirmed, apply the selected theme's colors and fonts consistently.
4. **Custom Theme**: If none of the pre-set themes work, follow the "Create your Own Theme" process.

## Create your Own Theme

If none of the themes work:
1. Generate a new theme with a descriptive name.
2. Define a cohesive color palette (primary, secondary, background, text).
3. Choose complementary font pairings.
4. Present for review before applying.
